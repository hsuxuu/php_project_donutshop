嗨老師！這次網頁我真的做好久（比上學期薑餅人還久）

這次的主題發想主要是因為我在甜甜圈店上班，
但是如果有客人想要跟我們訂甜甜圈的話，都只能用電話來溝通，非常耗時跟耗人力！
如果是大量的訂單，客人又不是很熟悉甜甜圈種類的話，來回溝通可能會花到半小時的時間QQ

所以這個主題就誕生了！

這次一樣是用Bootstrap做架構，基本上老師你看到的每字每句都是我自己寫的，
響應式應該主要算是Bootstap內建的，我只有自己修改版型，改成自己喜歡的樣子(❁´◡`❁)

這次一樣圖片都是自己做跟自己畫的（除了九顆甜甜圈是直接拿別人的之外）
logo是根據甜甜圈店的logo自己設計的，看起來應該很有一回事？
（帽子上的H是因為我姓許喔！X是因為XXX donut）

輪播跟吐司功能是主要Bootstrap提供的，一樣事後有再改成自己喜歡的樣子～
中間有一個傳值不換頁的功能ajax（在donut.php最下面）
是我去網路上找教學後改寫的（例如傳值完後要清除剛剛填寫的數量那些是自己加的）
